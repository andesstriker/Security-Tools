# Useful Commands

## Terminal commands
- `which `: tells you where a program is located
  - `which curl`
- `hexdump -C /bin/ls | less`: dumps all the hex in the file ?
- `cat`: concatenates files or reads standard input
- `;` can be used to write multiple commands in one line
- strace
  - ex `strace -f ./bomb` 
- `ltrace`
